Overview

Welcome to the documentation for the C++ CLI-based system. This software allows you to control various aspects of your system through a command-line interface.

Usage

Basic Commands

To get started, use the following commands:

# List all available commands
cli-tool --help

# Execute a command
cli-tool --run

API Reference

Function Signature

void executeCommand(const std::string& command);

Parameters

Parameter Type Description
command std::string The command to execute
